# Digital Image Processing Labs

Проект содержит лабораторные работы по цифровой обработке изображений и компьютерному зрению.

В рамках проекта реализованы:

* базовая обработка изображений;
* фильтрация и преобразование изображений;
* улучшение качества изображений;
* работа со свёрточными нейронными сетями;
* классификация изображений с использованием PyTorch.

---

# Используемые технологии

* Python
* NumPy
* OpenCV
* Matplotlib
* PyTorch
* Torchvision
* Jupyter Notebook

---

# Лабораторные работы

## Лабораторная работа №1

Реализована базовая обработка изображений и применение фильтров.

## Лабораторная работа №2

Реализованы методы преобразования и улучшения изображений.

## Лабораторная работа №3

Реализована дополнительная обработка изображений и визуальный анализ результатов.

## Лабораторная работа №4 — Классификация изображений MNIST с использованием PyTorch

Четвёртая лабораторная работа посвящена построению и обучению сверточной нейронной сети (CNN) для распознавания рукописных цифр из датасета MNIST.

### Основные этапы работы

#### Подготовка окружения

В проекте используются:

* PyTorch;
* Torchvision;
* DataLoader;
* Matplotlib.

Настраивается устройство выполнения:

* CUDA (GPU);
* MPS для Apple Silicon;
* CPU.

### Предобработка данных

Используется датасет MNIST из библиотеки Torchvision.

Для изображений применяется:

* преобразование в тензоры;
* нормализация значений пикселей.

Данные разделяются на:

* обучающую выборку;
* валидационную выборку;
* тестовую выборку.

Загрузка данных выполняется через `DataLoader` с batching и перемешиванием данных.

### Архитектура модели

Модель построена на основе сверточной нейронной сети.

Используются:

* сверточные слои (`Conv2D`);
* Batch Normalization;
* функция активации ReLU;
* MaxPooling;
* полносвязные слои (`Linear`);
* Dropout для борьбы с переобучением.

Архитектура позволяет эффективно извлекать признаки изображений и выполнять классификацию цифр.

### Обучение модели

Для обучения используются:

* функция потерь `CrossEntropyLoss`;
* оптимизатор `Adam`;
* несколько эпох обучения.

В процессе обучения:

* выполняется прямой проход;
* вычисляется ошибка;
* выполняется обратное распространение ошибки (`backpropagation`);
* обновляются веса модели.

### Валидация и тестирование

После каждой эпохи производится:

* вычисление средней ошибки;
* оценка точности модели;
* проверка качества классификации.

Отдельно реализовано тестирование модели на тестовой выборке.

### Визуализация

В ноутбуке выводятся:

* примеры изображений из датасета;
* предсказания модели;
* результаты классификации.

### Результат

В результате лабораторной работы реализована и обучена CNN-модель для классификации рукописных цифр. Проект демонстрирует базовый pipeline машинного обучения:

* подготовку данных;
* построение модели;
* обучение;
* валидацию;
* тестирование;
* анализ результатов.
